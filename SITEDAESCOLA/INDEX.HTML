    • Começando com a web

    Introdução à Web é uma série concisa que apresenta os aspectos práticos do desenvolvimento da Web. Você configurará as ferramentas necessárias para construir uma página da Web simples e publicar seu próprio código simples.
  • A história do seu primeiro site
    • Dá muito trabalho criar um site profissional, portanto, se você é novo no desenvolvimento da Web, recomendamos que comece pequeno. Você não vai construir outro Facebook imediatamente, mas não é difícil fazer seu próprio site online simples, então vamos começar por aí.
  • GUIAS
    • Ao trabalhar com os artigos listados abaixo, você passará do nada para obter sua primeira página online. Vamos começar nossa jornada!
  • Instalando o software básico
    • Quando se trata de ferramentas para construir um site, há muito por onde escolher. Se você está apenas começando, pode ficar confuso com a variedade de editores de código, estruturas e ferramentas de teste existentes. Neste artigo, mostraremos passo a passo como instalar o software necessário para iniciar o desenvolvimento básico da Web.
  • Como será o seu site?
    • Antes de começar a escrever o código do seu site, você deve planejá-lo primeiro. Quais informações você está apresentando? Quais fontes e cores você está usando? Aqui, descreveremos um método simples que você pode seguir para planejar o conteúdo e o design do seu site.
  • Lidando com arquivos
    • Um site consiste em muitos arquivos: conteúdo de texto, código, folhas de estilo, conteúdo de mídia e assim por diante. Ao construir um site, você precisa reunir esses arquivos em uma estrutura sensata e garantir que eles possam se comunicar. Este artigo explica como configurar uma estrutura de arquivo sensata para o seu site e quais problemas você deve estar ciente.
  • Noções básicas de HTML
    • HyperText Markup Language (HTML) é o código que você usa para estruturar seu conteúdo da web e dar a ele significado e propósito. Por exemplo, meu conteúdo é um conjunto de parágrafos ou uma lista de marcadores? Tenho imagens inseridas na minha página? Eu tenho uma tabela de dados? Sem sobrecarregá-lo, este artigo fornecerá informações suficientes para torná-lo familiarizado com o HTML.
  • Noções básicas de CSS
    • Cascading Style Sheets (CSS) é o código que você usa para estilizar seu site. Por exemplo, você deseja que o texto seja preto ou vermelho? Onde o conteúdo deve ser desenhado na tela? Quais imagens e cores de fundo devem ser usadas para decorar seu site? Neste artigo, mostraremos o que você precisa para começar.
  • Noções básicas de JavaScript
    • JavaScript é a linguagem de programação que você usa para adicionar recursos interativos ao seu site. Alguns exemplos podem ser jogos, coisas que acontecem quando botões são pressionados ou dados são inseridos em formulários, efeitos de estilo dinâmico, animação e muito mais. Neste artigo, daremos a você uma ideia do que é possível fazer com essa linguagem empolgante e como começar.
  • Publicando seu código de amostra
    • Depois de terminar de escrever o código e organizar os arquivos que compõem seu site, você precisa colocar tudo online para que as pessoas possam encontrá-lo. Este artigo descreve como obter seu código de amostra simples online com o mínimo de esforço.
  • Como a web funciona
    • Quando você acessa seu site favorito, muitas coisas complicadas acontecem em segundo plano e você pode não saber. Aqui, descreveremos o que acontece quando você visualiza uma página da Web em seu computador.

    Controlando a contagem da lista Às vezes, você pode querer contar de forma diferente em uma lista ordenada — por exemplo, começando de um número diferente de 1, ou contando para trás, ou contando em passos de mais de 1. HTML e CSS têm algumas ferramentas para ajudá-lo aqui. começar O startatributo permite iniciar a contagem da lista a partir de um número diferente de 1. O exemplo a seguir:
    1. Toast pita, leave to cool, then slice down the edge.
    2. Fry the halloumi in a shallow, non-stick pan, until browned on both sides.
    3. Wash and chop the salad.
    4. Fill pita with salad, hummus, and fried halloumi.

      • Invertido

      O reversedatributo iniciará a lista em contagem regressiva em vez de crescente. O seguinte exemplo:
      1. Toast pita, leave to cool, then slice down the edge.
      2. Fry the halloumi in a shallow, non-stick pan, until browned on both sides.
      3. Wash and chop the salad.
      4. Fill pita with salad, hummus, and fried halloumi.

      • Valor

      O valor atributo permite que você defina seus itens de lista para valores numéricos específicos. O seguinte exemplo:
      1. Toast pita, leave to cool, then slice down the edge.
      2. Fry the halloumi in a shallow, non-stick pan, until browned on both sides.
      3. Wash and chop the salad.
      4. Fill pita with salad, hummus, and fried halloumi.
